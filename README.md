# Dr Delplace — Dermatologue

- Preview GitHub Pages : https://fduthilleul.github.io/drdelplace.be/
- Domaine de production (à brancher) : https://drdelplace.be/
- RDV en ligne : [Progenda](https://progenda.be/calendars/docteur-delplace-daphne-dermatologue-ath-1cf29121-b19c-4393-abc8-a50a9ecb8f0e/availabilities?locale=fr)

## Développement local

```bash
git clone --recurse-submodules git@github.com:fduthilleul/drdelplace.be.git
cd drdelplace.be
hugo server
```
